HTML Language Codes

  HTML Country Codes  
  HTML Event Attributes  

HTML Language Codes are 2-character codes that specify a language.

For example, en for English, nl for Dutch, and ko for Korean.

Example

#

A lang attribute that specifies English (en) as the web page's language. 

<html lang="en">
...
</html>

Language Code Usage

A Language Code is a 2-letter abbreviation that specifies a language.

HTML uses ISO language codes to specify the language of a page (or element).

These language codes are used as values for the lang attribute.
