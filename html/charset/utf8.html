HTML UTF-8

  HTML CharSet ASCII  
  HTML CharSet Entities  

UTF-8 stands for 8-bit Unicode Transformation Format.

It‘s a character set with almost all known characters, punctuations, and symbols.

UTF-8 includes tens of thousands of characters that are used world-wide.

HTML5 uses UTF-8 as character encoding by default.

Example

#

Four UTF-8 characters from different languages/cultures: Latin, Arabic, Tamil, and Sinhala.

UTF-8 Character grouping

#

UTF-8 characters are organized in groupss, each with their own numeric range.
Below are some of the lower-range groups.

UTF-8 Character Group Numeric Range
UTF-8 Basic Latin 32 - 126
UTF-8 Latin Supplement 160 - 255
UTF-8 Latin Extended A 256 - 383
UTF-8 Latin Extended B 384 - 591
UTF-8 Modifier Letters 688 - 767
UTF-8 Diacritical Marks 768 - 879
UTF-8 Greek and Coptic 880 - 1023
UTF-8 Cyrillic 1024 - 1279
UTF-8 Cyrillic Supplement 1280 - 1327
