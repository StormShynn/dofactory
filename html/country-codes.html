HTML Country Codes

  HTML HTTP Status Codes  
  HTML Language Codes  

HTML Country Codes are 2-character codes that specify a country.

Examples include CA for Canada, BR for Brazil, and ES for Spain.

Example

#

A lang attribute that specifies English (en) as spoken in Great Britain (GB).

<html lang="en-GB">
...
</html>

Country Code Usage

A Country Code is a 2-character code that specifies a country.

HTML uses 5-character ISO codes according to this pattern: ll-CC.

The ll = lower-case language code, and the CC = upper-case country code.

These codes are used as the value for lang attribute.
