CSS Tables

  CSS Lists  
  CSS Hover  

An HTML table displays tabular data, just like a spreadsheet.

Tables have rows, columns, and, optionally, a header and a footer.

CSS can style tables with color, sizing, alignment, borders, and more.

Example

#

A <table> styled with CSS.

Firstname Lastname Country
Christina Berglund Sweden
Maria Larsson Sweden
Thomas Hardy UK 
<style>
  .tbl { border-collapse: collapse; width: 400px; }
  .tbl th, .tbl td { padding: 5px; border: solid 1px #bbb; }
  .tbl th { background-color: aliceblue; text-align: left; }
</style>

<table class="tbl">
  <tr>
    <th>Firstname</th>
    <th>Lastname</th>
    <th>Country</th>
  </tr>
  <tr>
    <td>Christina</td>
    <td>Berglund</td>
    <td>Sweden</td>
  </tr>
  <tr>
    <td>Maria</td>
    <td>Larsson</td>
    <td>Sweden</td>
  </tr>
  <tr>
    <td>Thomas</td>
    <td>Hardy</td>
    <td>UK</td>
  </tr>
</table>

Table Borders

#

By default, tables display without borders.

Borders can be added with the CSS border property.

A custom border added to the <table>, <th>, and <td> elements.

Firstname Lastname City
Maria Anders Berlin
Ana Trujillo México D.F.
Thomas Hardy London 
<style>
  .tb, .tb th, .tb td {
    border: 2px dashed orangered;
    padding: 5px;
  }

  .tb {
    width: 400px;
  }
</style>

<table class="tb">
  <thead>
    <tr>
      <th>Firstname</th>
      <th>Lastname</th>
      <th>City</th>
    </tr>
  </thead>
  <tbody>
    <tr>
      <td>Maria</td>
      <td>Anders</td>
      <td>Berlin</td>
    </tr>
    <tr>
      <td>Ana</td>
      <td>Trujillo</td>
      <td>México D.F.</td>
    </tr>
    <tr>
      <td>Thomas</td>
      <td>Hardy</td>
      <td>London</td>
    </tr>
  </tbody>
</table>

For details on borders, see our CSS border Property Reference.

Table Border Collapse

#

The border-collapse property specifies if table borders are collapsed or not.

Collapsed borders mean that rows and cells share the same border.

Possible values are: collapse and separate.

Two tables. One collapsed and the other separate.

Collapsed
Firstname Lastname Country
Christina Berglund Sweden
Maria Larsson Sweden
Thomas Hardy UK

Separate
Firstname Lastname Country
Christina Berglund Sweden
Maria Larsson Sweden
Thomas Hardy UK

Table Width and Height

#

The width and height properties sets the sizes of table, rows, and cells.

The property values can be any CSS length value, such as, px, cm, em, and other units.

A full-width table with custom heights for the table cells.

Firstname Lastname City
Maria Anders Berlin
Ana Trujillo México D.F.
Thomas Hardy London 
<style>
 .tbl-size { border-collapse: collapse; width: 100%; }
 .tbl-size th { background-color: aliceblue; text-align: left; }
 .tbl-size, .tbl-size td, .tbl-size th { 
   border: 1px solid #bbb; 
   padding: 5px; height: 45px; }
</style>

<table class="tbl-size">
  <thead>
   <tr> 
      <th>Firstname</th>
      <th>Lastname</th>
      <th>City</th>
   </tr>
  </thead>
  <tbody>
   <tr> 
      <td>Maria</td>
      <td>Anders</td>
      <td>Berlin</td>
   </tr>
   <tr> 
      <td>Ana</td>
      <td>Trujillo</td>
      <td>México D.F.</td>
   </tr>
   <tr> 
      <td>Thomas</td>
      <td>Hardy</td>
      <td>London</td>
   </tr>
  </tbody>
</table>

Horizontal Alignment

#

The text-align property sets the table content alignment.

It accepts these values: left, right, and center.

By default, <th> is center aligned and <td> is left aligned.

Right aligned table headers and cells.

Firstname Lastname City
Maria Anders Berlin
Ana Trujillo México D.F.
Thomas Hardy London 
<style>
 .tbl-align { border-collapse: collapse; width: 100%; }
 .tbl-align td, .tbl-align th { 
   border: 1px solid #bbb; 
   text-align: right; 
   padding: 5px; }
</style>

<table class="tbl-align">
  <thead>
   <tr> 
      <th>Firstname</th>
      <th>Lastname</th>
      <th>City</th>
   </tr>
  </thead>
  <tbody>
   <tr> 
      <td>Maria</td>
      <td>Anders</td>
      <td>Berlin</td>
   </tr>
   <tr> 
      <td>Ana</td>
      <td>Trujillo</td>
      <td>México D.F.</td>
   </tr>
   <tr> 
      <td>Thomas</td>
      <td>Hardy</td>
      <td>London</td>
   </tr>
  </tbody>
</table>

For details on text-align, see our CSS text-align Property Reference.

Vertical Alignment

#

The vertical-align property sets the content vertical alignment.

It accepts these values: top, bottom, and middle.

By default, the vertical alignment of table cells content is middle.

All table content is vertically aligned to bottom.

Firstname Lastname City
Maria Anders Berlin
Ana Trujillo México D.F.
Thomas Hardy London 
<style>
.tbl-vert { border-collapse: collapse; width: 100%; }
.tbl-vert td, .tbl-vert th {
  border: 1px solid #bbb;
  padding: 5px;
  height: 45px;
  vertical-align: bottom;
}
</style>

<table class="tbl-vert">
  <thead>
   <tr> 
      <th>Firstname</th>
      <th>Lastname</th>
      <th>City</th>
   </tr>
  </thead>
  <tbody>

   </tr>
   <tr> 
      <td>Thomas</td>

Table Padding

#

The padding property adds space between content and cell borders.

Padding can be added on each side or on all four sides.

All <th> and <td> elements have extra padding.

Thomas Hardy London

  text-align: left;

}

</style>

Hoverable Table

#

A mouse-over effect can be added to highlight table rows or columns.

Use the :hover selector on <tr> elements can accomplish this.

Mouse over the table to see the rows change color.

Firstname Lastname City
Maria Anders Berlin
Ana Trujillo México D.F.
Thomas Hardy London 
<style>
.tbl-hover {border-collapse: collapse; width: 100%;}
.tbl-hover th, .tbl-hover td {
  padding: 5px;
  text-align: left;
  border-bottom: 1px solid #ddd;
}
.tbl-hover tr:hover {
  background-color: #d5d2ff;
}
</style>

<table class="tbl-hover">
  <thead>
   <tr> 
      <th>Firstname</th>
      <th>Lastname</th>
      <th>City</th>
   </tr>
  </thead>
  <tbody>
   <tr> 
      <td>Maria</td>
      <td>Anders</td>
      <td>Berlin</td>
   </tr>
   <tr> 
      <td>Ana</td>
      <td>Trujillo</td>
      <td>México D.F.</td>
   </tr>
   <tr> 
      <td>Thomas</td>

Striped Table

#

A striped table or zebra effect can be created with CSS.

Use the :nth-child(even) selector to set the background color of alternating rows.

A table with alternating background colors.

Firstname Lastname City
Maria Anders Berlin
Ana Trujillo México D.F.
Thomas Hardy London 
<style>
.tbl-striped { border-collapse: collapse; width: 100%; }
.tbl-striped th, .tbl-striped td {
  text-align: left;
  padding: 5px;
}

.tbl-striped tr:nth-child(even) {
  background-color: #e4ebfb;
}

The color property sets the table text color.

Custom colors for table header and hovered rows.

Responsive Table

#

Tables can resize according to the browser size.

However, at some point the content does not fit the table cells anymore.

To make a table responsive, wrap it inside a <div> element with a scrollbar.

The overflow-x: auto property adds a horizontal scrollbar to the wrapper.

Shrink the browser to see that this table is responsive.

Firstname Lastname City
Maria Anders Berlin
Ana Trujillo México D.F.
Thomas Hardy London 
<style>
  .tbl-responsive { border-collapse: collapse; width: 700px; }
  .tbl-responsive th, .tbl-responsive td { 
    text-align: left; padding: 5px; 
    border: 1px solid #bbb; }
</style>

<div style="overflow-x: auto;">
  <table class="tbl-responsive">
    <thead>
      <tr>
        <th>Firstname</th>
        <th>Lastname</th>
        <th>City</th>
      </tr>
    </thead>
    <tbody>
      <tr>
        <td>Maria</td>
        <td>Anders</td>
        <td>Berlin</td>
      </tr>
      <tr>
        <td>Ana</td>
        <td>Trujillo</td>
        <td>México D.F.</td>
      </tr>
      <tr>
        <td>Thomas</td>
        <td>Hardy</td>
        <td>London</td>
      </tr>
    </tbody>
  </table>
</div>
