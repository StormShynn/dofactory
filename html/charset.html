HTML Charset

  HTML Reference  
  HTML CharSet ASCII  

A charset, or character set, is a set of character codes that are supported by HTML.

A browser must know what charset to use to correctly display a web page.

For HTML5, the standard character set is UTF-8.

A brief history of Character sets

#

ASCII was the first character encoding standard (i.e. Charset).

ANSI was the original Windows character set.

ISO-8859-1 was the default character set for HTML 4.

Today, UTF-8 is the default character set for HTML 5.

HTML charset Attribute

#

To inform the browser what character set to use, add a <meta> tag with a charset property.

This tag is placed in the <head> section on each page.

<meta charset="UTF-8">

Note:  If a web page starts with <!DOCTYPE html> (which indicates HTML5), then the above meta tag is optional, because the default for HTML5 is UTF-8.

The ASCII Character Set

#

ASCII was the first character encoding standard (i.e. Charset).

ASCII uses values 0 to 31 (and 127) for control characters.

ASCII uses values 32 to 126 for letters, digits, and symbols.

ASCII does not use values 128 to 255.

The ANSI Character Set (Windows-1252)

#

ANSI was the original Windows character set.

ANSI is identical to ASCII for values 0 to 127.

ANSI has a proprietary set of characters for values 128 to 159.

ANSI is identical to UTF-8 for values 160 to 255.

The ISO-8859-1 Character Set

#

ISO-8859-1 was the default character set for HTML 4.

8859-1 is identical to ASCII for values 0 to 127.

8859-1 does not use values 128 to 159.

8859-1 is identical to UTF-8 for values 160 to 255.

The UTF-8 Character Set

#

Today, UTF-8 is the default character set for HTML 5.

UTF-8 is identical to ASCII for values 0 to 127.

UTF-8 does not use values 128 to 159.

UTF-8 is identical to both ANSI and 8859-1 for values 160 to 255.

UTF-8 continues from value 256 and up with more than 10,000 different characters.

The @charset CSS Rule

#

The @charset CSS rule specifies the character encoding of a style sheet.
This rule is useful when using non-ASCII characters in some CSS properties, like content. 

@charset "UTF-8";

For details on the charset rule, see our CSS @charset Rule Reference.
