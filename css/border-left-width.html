CSS border-left-width Property

  CSS border-left-style  
  CSS border-radius  

border-left-width defines the width of the left border.

The width is set as a pixel value, or as thin, medium, or thick.

Example

#

A thick border-left-width.

An element with a thick left border 
<style>
  .border-left-thick {
    padding: 20px;
    width: 300px;
    border-left-style: solid;
    border-left-color: tomato;
    border-left-width: thick;
  }
</style>

<div class="border-left-thick">
  An element with a thick left border
</div>

Using border-left-width

#

The default border-left-width is medium or 1 pixel.

A border-left-style value is required for a border to show.

  border-left-style  
  border-radius  

Syntax

#

border-left-width: medium | thin | thick | 
length | initial | inherit;

Values

#

Value Description
medium Default. Medium border
thin Thin border
thick Thick border
length Sets border thickness using CSS units
initial Sets the value to its default value.
inherit Inherits the value from its parent element.

More Examples
