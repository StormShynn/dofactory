HTML Symbols

  HTML CharSet Entities  
  HTML Canvas  

HTML symbols are characters generally not found on a keyboard.

They include mathematical operators, greek letters, currencies, and more.

HTML entity notation ( &...; ) is used for these HTML symbols.

If no entity name is available for a symbol, then use the entity number or hex notation.

Example

#

Three symbol pairs created with entity name notation and entity number notation.
The symbols display the same.

∞ ∞ ∅ ∅ ∉ ∉

<p style='font-size:50px;'>&infin; &#8734; &empty; &#8709; &notin; &#8713;</p>
