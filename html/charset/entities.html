HTML Entities

  HTML CharSet UTF-8  
  HTML CharSet Symbols  

HTML entities display characters that are generally not available on a keyboard.

Entities are strings that start with an ampersand (&) and end with a semicolon (;),

An example is &frac14; which generates ¼.

Example

#

Some example characters that are not available on all keyboards.

€ ß ∮ ç ±
