CSS Color Names

  CSS Colors  
  CSS Backgrounds  

Color Names are values (names) that represent a color.

Examples includes turquoise, green, tomato, lightsalmon, and others.

These names can be used instead of HEX and RGB color values.

Example

#

An element with a named background color.

Moccasin 
<div style="background-color: moccasin; padding: 30px;">
  Moccasin
</div>

Example

#

An example that uses 3 color names to style an element.

  • An aliceblue background for the container
  • A lightblue border on the container
  • A mediumblue text color
An element styled with color names 
<div style="background-color:aliceblue;
            border:5px solid lightblue;
            color:mediumblue;
            padding:30px;">
  An element styled with color names
</div>
