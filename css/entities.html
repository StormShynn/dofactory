CSS Entities

  CSS Length  
  CSS Functions  

CSS entities display special characters in an HTML document.

These are characters that are generally not available on keyboards.

In CSS they are commonly used with the content property.

Example

#

A trademark character displayed after a paragraph element.

Important brand name 

<style>
  p.my-p:after {
    content: ' \2122';
  }
</style>

<p class="my-p">Important brand name </p>
